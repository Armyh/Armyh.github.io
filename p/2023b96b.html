国内的chat AI-星火大模型的介紹

中国 1.2k 4 min

星火大模型

https://xinghuo.xfyun.cn/

简介

  • 讯飞星火认知大模型是一款由中国科技公司科大讯飞研发的人工智能产品。它具备强大的自然语言处理能力和智能对话系统，可以与人类进行自然交流并提供多种认知智能服务。讯飞星火认知大模型在各个领域都有着广泛的应用，为用户提供了高效、准确的解决方案。它的出现不仅推动了人工智能技术的发展，也为人们的生活带来了便利和创新。

使用体验

  • 在网站里面选择登录,点击导航栏 SparkDesk 就可以使用了，上手很快。
  • 还可以根据你的形容词 AI 生成图片、PPT 生成、图片识别等能力，点击聊天框上面的，选择插件一栏的插件即可。
  • 除了 SparkDesk 功能以外，还具备了插件中心，就是专项性的知识咨询，功能挺齐全的。下面是简历插件的体验

  • 更多有趣功能等你来体验！

星火大模型 API 接口调用

  • 其实讯飞开放平台对于开发者来说是真的很恩惠的。

  • 个人实名认证可以免费领取 200 万 tokens/年，注意是 V3.0 套餐、V2.0 套餐、V1.5 套餐分别送 200 万 tokens，也就是 600 万 tokens，这完全够一个个人网站使用一整年了。

  • 接入 API，在右下角你可以选择你的模块，有 SDK 接入还有 web 接入。这里就以 web 接入为例

  • 点击 web 文档，滑到最最最下面可以看到二级标题：3.调用示例

  • 点击 js 调用示例下载，然后打开 vscode

准备

  • 在 demo 中填写==APPID== 、==APISecret== 、==APIKey== ，可到控制台-我的应用-大模型页面获取
  • 安装 nodejs

  • 打开 cmd，进入 demo 目录，执行如下命令
  • 在 chrome 浏览器中打开本机 IP 地址，可打开 cmd 输入 ipconfig 获取，例如

  • 打开网址之后就可以看到效果了，如果想要根据自己的业务去改动，还得去看代码，制定修改。我下次会在本博客里面接入星火大模型的API，并且接入附上教程。

国内其他厂商大模型

360智脑文心一言同义千问书生浦语百川智能

国内的AI发展虽然起步较晚，但是也有不少的厂商在这一方面下==功夫== 。例如，360智脑、文心一言、通义千问、书生浦语、百川智能等等，都在AI方面努力追赶国际先进水平。这些公司致力于研发和应用人工智能技术，推动着中国在人工智能领域的发展。并且在一些领域已经取得了令人瞩目的成果。例如：人脸识别技术、语音识别技术、自然语言处理技术、医疗影像诊断、自动驾驶技术等人工智能领域。相信随着技术的不断进步和应用的深入推广，中国的人工智能产业将迎来更加==广阔== 的发展前景。

总结

我体验过国内的多种AI大模型，感觉这些相比其它厂商要好很多，比如关于文案的写作方面（估计吃了不少的好文章），还有就是它拥有实时同步的能力（你询问最近几天发生的什么，它都可以回答出来）。
